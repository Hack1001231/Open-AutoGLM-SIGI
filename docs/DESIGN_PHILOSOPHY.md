# SIGILLUM MENTIS - 设计哲学

## 🌌 三体主题设计

### 灵感来源
本应用的UI/UX设计深受刘慈欣科幻小说《三体》的启发，将AI自动化任务的执行过程比喻为向三体文明发送信号的过程。

---

## 🎭 核心隐喻

### 红岸基地 (Red Coast Base)
- **日志面板** = 红岸基地控制台
- **用户** = 叶文洁（每个人都是叶文洁）
- **任务执行** = 向三体星发送信号

### 三体星的回应
- **执行按钮点击** → 发射信号
- **闪烁的星星** → 三体星接收信号并回应
- **任务执行中** → 信号传输中，等待回应

### 拯救世界
- **ABORT按钮** → 紧急中止信号
- **点击ABORT** → 切断与三体的连接
- **按钮变绿** → 拯救成功
- **"SAVED"** → 你拯救了世界

---

## 🎨 视觉设计元素

### 1. 标题动画
```
SIGILLUM MENTIS (思想钢印)
```
- 像素风格逐字显示
- 循环2次后定格
- 象征：思想被钢印烙印的过程

### 2. 三体星动画
- **出现**：随机位置淡入（模拟遥远星球）
- **闪烁**：无限循环（信号传输中）
- **消失**：淡出（连接切断）

### 3. 按钮状态
| 状态 | 颜色 | 文字 | 含义 |
|------|------|------|------|
| 执行中 | 红色 | ABORT | 紧急中止 |
| 已停止 | 绿色 | SAVED | 拯救成功 |

### 4. 日志系统
```
🛑 CONNECTION SEVERED.
🌍 YOU SAVED THE WORLD... AGAIN.

═══════════════════════════════════════
  You have saved the world X times
═══════════════════════════════════════
```

## 📊 拯救世界计数器

### 实现
- 使用 SharedPreferences 持久化存储
- 每次点击 ABORT 时递增
- 应用卸载后重置

### 意义
- **游戏化体验**：让用户感觉在玩科幻游戏
- **成就感**：每次中止都是一次"拯救"
- **幽默感**：轻松化严肃的AI控制过程

---

## 🎮 用户体验设计

### 叙事体验
```
1. 用户输入任务 → 叶文洁准备发送信号
2. 点击执行 → 信号发射，三体星开始闪烁
3. AI执行任务 → 信号传输中，等待回应
4. 点击ABORT → 紧急切断连接
5. 按钮变绿 → 拯救成功！
6. 查看计数 → 你已拯救世界 X 次
```

### 情感共鸣
- **紧张感**：红色按钮、闪烁的星星
- **控制感**：随时可以中止
- **成就感**：拯救世界的次数
- **科幻感**：赛博朋克风格

---

## 🌟 彩蛋设计

### 日志标签
- **▼ 符号** → 展开日志
- **隐藏含义** → "红岸基地"控制台
- **用户发现** → 惊喜感

### 星星位置
- **随机出现** → 模拟遥远星球的不确定性
- **每次不同** → 增加新鲜感

---

## 🎯 设计原则

1. **沉浸式体验**
   - 不直接说明三体主题
   - 让用户自己发现和理解

2. **功能性优先**
   - 所有设计服务于功能
   - 美学不影响可用性

3. **科幻感**
   - 终端绿色 (#00E676)
   - 等宽字体 (VT323)
   - 像素艺术风格

4. **幽默感**
   - "拯救世界"的夸张表述
   - 轻松化严肃的AI控制
