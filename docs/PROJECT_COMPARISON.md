## 🤖 AI Agent 对比

### Hybrid: phone_controller.py
```python

    MODE_ACCESSIBILITY = "accessibility"

    def screenshot(self)      # 截图

    def swipe(...)            # 滑动
    def input_text(text)      # 输入
    
    # 每个操作都有两种实现：
    # _xxx_accessibility()  - 无障碍服务
    # _xxx_ladb()           - LADB 备用
```

```python
class SimplePhoneAgent:

    def _parse_action(self, content)    # 解析 AI 响应
    def _scale_coordinates(x, y)        # 坐标缩放
    def _check_repeated_action(...)     # 死循环检测
    def request_graceful_stop(...)      # 优雅停止
```

### 3️⃣ ⚡ 性能与稳定性：从实验到生产

#### Hybrid（原始版）
```
跨进程通信架构：
Termux 进程 ←→ HTTP ←→ Helper APP 进程
    ↓                        ↓
  可能崩溃              可能崩溃
    ↓                        ↓
  需要重启              需要重启
```
**问题：**
- ❌ 双进程不稳定
- ❌ 通信延迟高（~50ms）
- ❌ 内存占用大（~150MB）
- ❌ 容易出现进程同步问题

#### SIGILLUM MENTIS（当前版）
```
单进程集成架构：
┌────────────────────────────┐
│   SIGILLUM MENTIS 进程      │
│  Python + Kotlin 一体化     │
│  进程内 HTTP（~3ms）        │
└────────────────────────────┘
```

**性能提升数据：**

| 指标 | Hybrid | SIGILLUM MENTIS | 提升 |
|------|--------|-----------------|------|
| **通信延迟** | ~50ms | ~3ms | **16x 更快** |
| **内存占用** | ~150MB | ~80MB | **节省 47%** |
| **CPU 占用** | ~15% | ~8% | **节省 47%** |
| **启动时间** | ~10s（需启动两个APP） | ~3s | **3x 更快** |
| **崩溃率** | 中等（双进程） | 低（单进程） | **更稳定** |
| **响应速度** | ~200ms/操作 | ~100ms/操作 | **2x 更快** |

**成果：**
- ✅ **快**：16 倍通信速度提升
- ✅ **稳**：单进程架构，崩溃率大幅降低
- ✅ **省**：内存和 CPU 占用减半
- ✅ **顺**：操作响应速度翻倍
